Arrays

Arrays

Estrutura que nos permite criar lista de valores.

Criação de Arrays

Utilize "[]" couchetes. Os elementos dentro do Array devem ser separados por vírgulas.

Trabalhando com Arrays

push Adiciona um novo elemento ao final do Array.
unshift Adiciona um novo elemento no primeiro índice do Array.
pop Remove o último item do Array.
shift Remove o primeiro item do Array.
splice Pode remover ou adicionar um item no Array. Indique a posição do início da operação, quantos itens quer remover e os itens que quer adicionar.
length Retorna o tamanho do Array, (também pode ser adicionado em strings)
[Índice] Permite acessar um elemento do Array de acordo com sua posição (também pode ser usado em String)

Declarando várias variáveis com Arrays

Você pode declarar variáveis que recebem valores dos elementos do Array.

                    
                        let [primeiroElemento, segundoElemento] = [1,2,3]

Operador Spread (...)

Serve para espalhar os elementos de um Array.
