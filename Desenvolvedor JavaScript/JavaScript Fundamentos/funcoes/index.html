Funções

Apresetação

As linguagens de programação possuem estruturas que nos permitem reutilizar códigos, essas estruturas são as funções.

Isso torna nosso código mais organizado e noso trabalho mais simples.

O que são as Funções?

Imagine que você queira imprimir a tabuada de algum número, como fizemos anteriormente.

Porém, agora você terá que fazer em vários lugares do seu programa, com diferentes números. Imagine o trabalho de ter que copiar e colar o mesmo código em vários lugares! E se fosse uma conta mais complexa e você tivesse que alterar algo na lógica? Daria muito mais trabalho arrumar em vários lugares.

Essa é uma das razões de termos o que chamamos de Funções. Funções são basicamente um conjunto de linhas de código que podemos chamar a qualquer hora e em qualquer lugar.

Então, podemos criar uma função que faça essa tabuada para nós e chamá-la onde precisarmos.

Um ótimo jeito de deixar o seu código mais limpo e fácil de manter é sempre criar funções que tenham apenas uma responsabilidade, ou seja, não devem fazer mais de uma coisa. Também tente fazer com que a função não tenha mais de 20 linhas. Se tiver mais de 20 linhas, provavelmente ela está fazendo mais de uma coisa.

Caso em algum momento você pense em copiar um código e colar em outro lugar, provavelmente você estará precisando de reutilização de código, indicando que provavelmente você está precisando criar uma função para evitar código repetido.

Para declararmos nossas funções, usamos a palavra chave: function seguida do nome da função e parênteses, que pode ter argumento ou não.

Ex.: 

                        
                            function somar(let a, b) {
                                return a + b;
                            }

                            console.log(somar(2,2));

                            //SAÍDA:  4

E para chamar a função, apenas declaramos o nome da função seguida com os parênteses '()' como fiz acima em console.log.

Variáveis locais e globais

Variável Local

Ela só existe onde foi declarada;
Pode ser usada em seu escopo local, como funções e laços;

Variável Global

Ela só existe em qualquer parte do código; pode ser declarada com window.variavel;
ou no NodeJs com global.variavel;

Funções Anônimas e Callbacks

No javaScript funções podem ser armazendas em variáveis. Esse comportamento nos desobriga a nomear uma função;

Esse tipo de função é chamada de Anônima 

                        
                            let minhaFuncao = function(let a, b) {
                                return a + b;
                            }

                            console.log(minhaFuncao(2,2));

                            //SAÍDA:  4

CallBack

É uma função que será executada posteriormente. Muito utilizado em assincrono; 

                            
                                console.log('começou');

                                // EXECUTA NO CONSOLE A STRING 'COMEÇOU'

                                setTimeout(somar, 2000); // VAI CHAMAR A FUNÇÃO SOMAR, APOS 2S

                                // USANDO FUNÇÃO ANÔNIMA

                                setTimeout(function(){
                                    console.log('passaram 2 segundos');
                                },2000)

Arrow Functions

São funções criadas no formato "() => {}". Caso tenha apenas um parâmetro, não é necessário usar parênteses. Caso tenha apenas uma linha, não é necessário usar chaves e nem "return".
