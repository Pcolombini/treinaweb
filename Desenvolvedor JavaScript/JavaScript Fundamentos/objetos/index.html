Nesta aula vamos conhecer os objetos, como começar a criá-los e finalmente para que servem os Symbols.

O que são objetos? Agrupando propriedades e valores

Objetos são tipos de dados que possuem mais funcionalidades e são mais complexos que os tipos primitivos como os tipos string e number.

São basicamente um modo de agrupar valores.

Podemos pensar neles como objetos que existem no mundo real. Como um lápis por exemplos, que possue caracteristicas e coisas que podemos fazer com eles.

Os objetos servem para agrupar variáveis e funções, e permitem o reuso dessas caracteristicas e métodos.

Objetos JavaScript

No JavaScript criamos objetos quase como criamos arrays, que são da mesma família, arrays são objetos, mas os objetos usamos chaves {} e os arrays couchetes [].

Por exemplo o lápis, vamos criar um objeto: 

                    
                        // Criando o objeto

                        const lapis = {
                            // propriedade chave: valor
                            cor: 'Preto',
                            tamanho: 15,

                            /* funções escrita de forma simples,
                             funções podem ser escritas como 
                             nome: function () {} ou arrow nome: () => {} */

                            escrever() {
                                console.log('Escrevendo...');
                            },

                            apagar:function(){
                                console.log('Apagando...');
                            },

                            rabiscar: () => {
                                console.log('Rabiscando...');
                            }
                        };

                        // Acessando os atributos e funções

                        lapis.cor;
                        lapis['cor'];

                        let cor = 'cor';

                        lapis[cor];

                        // Criando novos atributos

                        lapis.corDaBorracha = 'Rosa';
                        lapis['Ponta'] = 1.5;

                        // Removendo atributos

                        delete lapis.Ponta;

                        /* Para saber o nome de
                         todas as chaves que um objeto possue */

                        console.log(Object.keys(lapis));
