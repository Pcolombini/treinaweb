Objetos - Agrupando valores

Apresentação

Muitas vezes também vamos precisar agrupar vários tipos de dados em certas estruturas, que chamamos de Objetos.

Nesta aula vamos conhecer os objetos, como começar a criá-los e finalmente para que servem os Symbols.

Também vamos conhecer o JSON, um dos formatos mais importantes para a troca de dados entre softwares.

O que são objetos? Agrupando propriedades e valores

Objetos são tipos de dados que possuem mais funcionalidades e são mais complexos que os tipos primitivos como os tipos string e number.

São basicamente um modo de agrupar valores.

Podemos pensar neles como objetos que existem no mundo real. Como um lápis por exemplos, que possue caracteristicas e coisas que podemos fazer com eles.

Os objetos servem para agrupar variáveis e funções, e permitem o reuso dessas caracteristicas e métodos.

Objetos JavaScript

No JavaScript criamos objetos quase como criamos arrays, que são da mesma família, arrays são objetos, mas os objetos usamos chaves {} e os arrays couchetes [].

Por exemplo o lápis, vamos criar um objeto: 

                    
                        // Criando o objeto

                        const lapis = {
                            // propriedade chave: valor
                            cor: 'Preto',
                            tamanho: 15,

                            /* funções escrita de forma simples,
                             funções podem ser escritas como 
                             nome: function () {} ou arrow nome: () => {} */

                            escrever() {
                                console.log('Escrevendo...');
                            },

                            apagar:function(){
                                console.log('Apagando...');
                            },

                            rabiscar: () => {
                                console.log('Rabiscando...');
                            }
                        };

                        // Acessando os atributos e funções

                        lapis.cor;
                        lapis['cor'];

                        let cor = 'cor';

                        lapis[cor];

                        // Criando novos atributos

                        lapis.corDaBorracha = 'Rosa';
                        lapis['Ponta'] = 1.5;

                        // Removendo atributos

                        delete lapis.Ponta;

                        /* Para saber o nome de
                         todas as chaves que um objeto possue */

                        console.log(Object.keys(lapis));

Conhecendo o JSON e sua importância

JSON significa "JavaScript Object Notation", que traduzido é "Notação de objeto JavaScript".

JSON é um padrão para escrever objetos com a sintaxe do JS. É suportado por diversas linguagens de programação, possue o objetivo de transferir dados de uma linaguagem para a outra.

Como JSON trata de um objeto que vai ser transferido por mensagem, alguns coisas não são compatíveis, as funções e objetos complexos como o Date(), não são passados. Podem ser passados por JSON os tipos primitivos, alguns objetos e arrays.

Para escrever JSON também usamos chaves {}, mas as propriedades precisam estar envolvidas em aspas duplas "".

Para transferir um objeto JavaScript para JSON e assim transferir para outros programas precisamos transformálo em uma string, com a função tringfy do objeto JSON:
JSON.stringfy(objeto);.

E quando recebemos strings e precisamos transforma em objetos novamente usamos uma função dentro do objeto JSON que é o parse:
JSON.parse("{"cor":"Preto","tamanho":15,"corDaBorracha":"Rosa"}");

Onde o JSON pode ser usado?

Vimos um pouco sobre a importância do JSON. Agora vamos entender onde ele costuma ser mais usado.

Transferência de Dados

O JSON é o formato mais utilizado hoje em dia quando precisamos que um software se comunique com outro.

Quando você envia uma mensagem para um amigo, o aplicativo envia esse dado como JSON para o servidor da empresa responsável pelo aplicativo, processa e envia o JSON para o Smartphone do seu amigo.

Quando acessa o YouTube, vários dados sobre os vídeos chegam ao seu navegador ou aplicativo por JSON.

E não é apenas entre softwares da mesma empresa! Empresas também podem fornecer serviços que fornecem dados úteis para que terceiros possam criar outros softwares!

Um bom exemplo é o ViaCep, que nos retorna um JSON com as informações de um endereço a partir de um CEP informado. Sabe quando você digita o CEP em um formulário e ele já preenche os outros campos? Eles vieram de um serviço como o ViaCep através de JSON.

No link abaixo, substitua "01001000" pelo seu CEP e você verá um JSON com as informações como rua, bairro, cidade, etc.
https://viacep.com.br/ws/01001000/json/

Configurações

O JSON também pode ser usado como um arquivo onde armazenamos informações que serão lidas por um programa para ele saber como se comportar, ou seja, suas configurações.

O próprio VS Code possui várias configurações que você pode customizar editando um arquivo JSON de configuração que ele possui.

Projetos de Front-End modernos e de Node.js costumam ter um arquivo chamado package.json, onde armazenamos informações sobre o nome do projeto, sua versão e várias outras informações importantes para o projeto.

Para saber mais, você pode conferir nosso curso de NPM.

Armazenamento de Dados

O JSON também pode ser usado para armazenar dados. Assim como podemos ter o JSON em uma variável, também podemos ter ele salvo em um arquivo. Para pegar o JSON, basta ler o arquivo.

É exatamente assim que o Tw Dev Server simula um banco de dados.

Também existe um banco de dados chamado MongoDB. Ele utiliza uma estrutura baseada no JSON para armazenar seus dados.
