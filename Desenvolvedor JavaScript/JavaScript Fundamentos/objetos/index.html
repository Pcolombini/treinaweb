Apresentação

Muitas vezes também vamos precisar agrupar vários tipos de dados em certas estruturas, que chamamos de Objetos.

Nesta aula vamos conhecer os objetos, como começar a criá-los e finalmente para que servem os Symbols.

Também vamos conhecer o JSON, um dos formatos mais importantes para a troca de dados entre softwares.

O que são objetos? Agrupando propriedades e valores

Objetos são tipos de dados que possuem mais funcionalidades e são mais complexos que os tipos primitivos como os tipos string e number.

São basicamente um modo de agrupar valores.

Podemos pensar neles como objetos que existem no mundo real. Como um lápis por exemplos, que possue caracteristicas e coisas que podemos fazer com eles.

Os objetos servem para agrupar variáveis e funções, e permitem o reuso dessas caracteristicas e métodos.
